Town Sounds

As a musician, I always thought it would be cool if you could find local live music on an interactive map. I built a web app to try and accomplish this. Using python, I automated the webscraping of a popular live music website in order to get concert event information. I then populated the map with this data. The resulting web app allowed users to search for live music in their town and view details about the event: the artist, genre, performance venue, time/date, and other pertinent details. I also implemented several filters for the user to narrow down their live music search: a music genre filter, as well as a time filter to search events by specific time periods.

Unfortunately, I had to discontinue the app due to a number of challenges: Amazon Web Services (AWS) storage costs and difficulty obtaining data (the live music website I was using a data source did not have an API to streamline consumption of data into the map). Still, I learned a ton of cool new skills during this project, including: AWS, webscraping with Python's BeautifulSoup library, web browser automation with Python's Selenium library, plus a whole lot of JavaScript.
